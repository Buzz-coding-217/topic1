Activities
Activity 1
In the first activity, we were asked what is IoT Applications with examples and what are the main features of it. We were able to understand what an IOT application is and as a group we came up with a number of examples like smart watches. We were also able to give some main features about IOT applications as well.
Activity 2
In the second activity, we were asked about building a smart building along with its some features. As a group, we were thinking of multiple scenarios where we can apply IoT application, along with their features like what sensors will be used or how the device with integrate with one another
Activity 3
In the third activity, we were asked about the control interface of the smart building, like some high level features, requirements and users of the building. we analyse this in detail, we were able to give sufficient features to the building, along with variety of users and parts required to make the system
