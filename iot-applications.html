Activities
Activity 1
In the first activity, we were asked what IoT Applications are, with examples, and what the main benefits are. We were able to define an IOT application and came up with several examples, such as smart watches, as a group. We were also able to provide some key features of IOT applications.

Activity 2
In the second activity, we were asked about constructing a smart building and its various features. As a group, we discussed various scenarios in which we could use IoT applications, as well as their features such as which sensors would be used and how the devices would interact with one another.

Activity 3
In the third activity, we were asked about the smart building's control interface, including some high-level features, requirements, and building users. We thoroughly examined this, and we were able to provide sufficient features to the building, as well as the variety of users and parts required to make the system work.
